# Conecta UERJ 2026

Microsite institucional desenvolvido para divulgação do **Conecta UERJ – Mostra de Estágios 2026**.

O objetivo do projeto é oferecer uma experiência moderna, responsiva e acessível para estudantes, empresas e visitantes interessados no evento.

---

## Objetivos

- Divulgar o evento de forma clara e atrativa;
- Apresentar empresas participantes;
- Disponibilizar programação completa;
- Informar localização e estrutura do evento;
- Reunir informações úteis em um único ambiente digital.

---
